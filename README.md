# ithings
 ithings is short for Iot things

# how to build
```
$	make  ## build the project
$   make clean ## clean the project.
```

# how to run
1. edit the config.yaml in conf/ directory or env ${QENV_CONFIG_PATH}/conf/
2. run ithings in linux (ithings.exe in windows)
